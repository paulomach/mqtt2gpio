# mqtt2gpio

Very crude mqtt to gpio translator.

## Motivation

Personal raspberrypi home automation project.

### Notes

4x power relay shield configuration
* pin 19 - channel 1
* pin 26 - channel 2
* pin 20 - channel 3
* pin 21 - channel 4
